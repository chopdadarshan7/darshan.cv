## Overview

This project is a responsive portfolio and CV website built for **Darshan Chopda**, a **Full Stack Developer** and **AI Prompt Engineer Specialist**.

The website highlights:

- professional summary
- technical skills across frontend, backend, AI, and tools
- work experience and internships
- featured projects with GitHub and live demo links
- contact and social links

It is designed as a lightweight static website that is fast to load, easy to deploy, and simple to customize.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/chopdadarshan7/darshan.cv.git
cd darshan.cv
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm start
```

Then open:

`http://localhost:3000`

---

## Customization

Most of the portfolio content is managed from:

[`js/script.js`](./js/script.js)

You can update:

- name and title
- about section
- skills list
- experiences
- featured projects
- GitHub, LinkedIn, email, and resume links

For layout and styling changes, edit:

- [`index.html`](./index.html)
- [`css/styles.css`](./css/styles.css)
